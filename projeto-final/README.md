# 🤖 Entendendo o Desafio

Agora é a sua hora de praticar, aprender fazendo e criar um projeto para o seu portfólio! Neste Lab, você vai construir um **Assistente Virtual Com Inteligência Artificial**. A ideia é criar uma solução simples, capaz de conversar com uma pessoa usuária, entender uma necessidade e responder com base em informações organizadas.

Antes de começar, acesse o Repositório Base, pois ele contém todo o passo a passo documentado do desafio:
* 🔗 [Repositório Base: Construa Seu Assistente Virtual Com Inteligência Artificial](https://github.com/digitalinnovationone/dio-lab-bia-do-futuro)

Você também pode consultar o Repositório de Exemplo, usado como apoio nas partes práticas das videoaulas:
* **Repositório De Exemplo:** Edu, Educador Financeiro Inteligente

    * 🔗 [Edu: Educador Financeiro Inteligente](https://github.com/falvojr/dio-lab-bia-do-futuro)

> 💡 **Nota:** O projeto de referência apresenta um Agente Financeiro Inteligente, mas você pode adaptar a ideia para outro contexto. O mais importante é mostrar que seu assistente tem um objetivo claro, usa uma base de conhecimento e responde de forma útil.

---

## 🎯 O Que Criar

Seu assistente deve ajudar uma pessoa em uma tarefa específica. Ele pode, por exemplo:

* **Entender** uma dúvida ou necessidade;
* **Usar** informações de uma base de conhecimento;
* **Responder** de forma simples e clara;
* **Evitar** respostas inventadas (alucinações);
* **Dizer** quando não tiver informação suficiente;
* **Ajudar** a pessoa usuária a tomar uma próxima decisão.

*Não se preocupe em criar uma solução perfeita. O foco é criar um protótipo simples, funcional e bem explicado.*

---

## 🚀 Como Fazer

O Repositório Base organiza o desafio em **6 passos**. As aulas explicam cada um deles com mais detalhes, então vale revisá-las sempre que precisar.

1. **Documentação:** Explique o que seu assistente faz, para quem ele serve e como deve se comportar.
2. **Base de Conhecimento:** Organize as informações que o assistente vai usar para responder melhor.
3. **Prompts:** Escreva as instruções que orientam as respostas da IA.
4. **Aplicação Funcional:** Crie uma forma simples de testar a conversa com o assistente.
5. **Avaliação e Métricas:** Show como você avaliou se as respostas fazem sentido.
6. **Pitch:** Apresente o problema, a solução e o valor do seu projeto.

> 📁 A base de conhecimento pode começar pequena. Use os dados do repositório como ponto de partida e adapte as informações conforme o problema que seu assistente vai resolver.

## 🤖 Use o DIO Agent Como Apoio

Durante o desafio, você também pode usar o DIO Agent como um parceiro de estudos. Ele pode te ajudar a entender o desafio, organizar as ideias, melhorar os prompts, revisar o README e destravar dúvidas durante o desenvolvimento.

Você pode começar com um prompt simples como este:

```text
Estou fazendo o Lab "Construa Seu Assistente Virtual Com Inteligência Artificial".

Repositório do desafio: https://github.com
Meu tema é: <preencha aqui>

Me ajude a organizar os 6 passos do projeto:
- Documentação do agente;
- Base de conhecimento;
- Prompts do agente;
- Aplicação funcional;
- Avaliação e métricas;
- Pitch final.

Importante: não quero uma resposta pronta para copiar. Quero entender o processo e construir minha própria solução.
```

⚠️ *Use o DIO Agent como apoio, mas lembre-se: o projeto final deve refletir suas escolhas, seus testes e seu aprendizado.*

## 📦 O Que Entregar

Dê um **Fork** no Repositório Base ou crie seu próprio repositório no GitHub, organizando seus arquivos da forma que preferir. Uma estrutura recomendada de pastas é:

```text
assistente-virtual-ia/
├── data/
├── docs/
├── src/
└── README.md
```

* **`README.md`**: Use para apresentar o escopo do seu projeto.
* **`data/`**: Pasta destinada a armazenar sua base de dados/conhecimento.
* **`docs/`**: Pasta para guardar a documentação das etapas e guias de prompts.
* **`src/`**: Pasta reservada para os códigos fontes da aplicação.

*Se ainda estiver começando, tudo bem. Entregue uma primeira versão simples, mas tente deixar claro como você passou pelos 6 passos do desafio.*
